# Quick PoC for Loading Mitsuba Rendered Scenes using Three.js

## Prerequisites

* Node.js (Only tested on v21.2.0)
* NPM (Only tested on v10.2.3)

## How to Run

### Install Dependencies
```bash
# npm i
```

### Run the Server
```bash
# npx vite
```

### Open the Browser & Navigate to the URL

```
http://localhost:5173/
```
